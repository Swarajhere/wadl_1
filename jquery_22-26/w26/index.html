B.E. Course Structure

First Year (FE)

Engineering Mathematics I: Covers calculus, differential equations, and linear algebra for engineering applications.

Engineering Physics: Introduces fundamental physics concepts like mechanics, optics, and thermodynamics.

Basic Electrical Engineering: Basics of electrical circuits, systems, and applications.

Second Year (SE)

Data Structures: Fundamentals of data structures like arrays, linked lists, stacks, and queues.

Discrete Mathematics: Topics like set theory, logic, and graph theory for computer science.

Digital Electronics: Introduction to digital circuits, logic gates, and microprocessors.

Third Year (TE)

Database Management Systems: Concepts of databases, SQL, and data modeling.

Operating Systems: Study of OS concepts like processes, memory management, and file systems.

Software Engineering: Principles of software development, testing, and project management.

Final Year (BE)

Machine Learning: Introduction to ML algorithms, models, and applications.

Cloud Computing: Concepts of cloud architecture, services, and deployment.

Project Work: A capstone project to apply engineering knowledge to a real-world problem.
